Resume

Code

This resume is scrapped from GitHub, and displayed in the form of a png. The png is rendered with ImageMagick from a pdf as part of my compilation process, but the real magic is behind the png.

The resume source is a JSON document -- it's exact schema is ill-defined, since I'm the only one using it and it changes to suit my needs. That JSON is rendered to LaTeX source file via ruby ERB templating (load from JSON, scrub to texify things like ampersands and underscores, and run ERB), and then handed off to pdflatex to do its wizardry. There is a Makefile which exists to pass arguments to the ruby script, but does little else.

Please feel free to use the source either as a jumping-off point for your own resume templating needs, or use the generated LaTeX source as a template for your own Resume or CV.

The ultimate goal of these templating shenanighans is to produce HTML output which could be embedded and styled on this page. I will begin work on this feature at a later date.
